Tentang Saya

Saya adalah seorang Web Developer dengan ketertarikan besar pada pengembangan antarmuka yang bersih, responsif, dan mudah diakses. Fokus utama saya adalah membangun tampilan frontend yang tidak hanya fungsional tetapi juga menyenangkan secara visual.

Di sisi lain, saya juga memiliki kemampuan dalam desain digital dan pembuatan konten kreatif — mulai dari website portofolio, landing page, hingga visual untuk media sosial. Saya terbiasa bekerja menggunakan HTML, CSS, JavaScript, serta alat bantu seperti Figma dan Canva.

Saya percaya bahwa setiap detail dalam tampilan web harus dapat menciptakan pengalaman yang nyaman, inklusif, dan bermakna bagi penggunanya.
